考虑到您的需求，我推荐使用 多服务混合 的方案：

✅ 优点：
  • 完全免费
  • 多种样式可选
  • 支持统计报告

❌ 缺点：

🔧 推荐的混合集成方案

结合多种服务的优势，为不同环境提供最佳体验：

function initSmartCounter() { const isVercel = window.location.hostname.includes('vercel.app'); const isGitHub = window.location.hostname.includes('github.io'); if (isVercel) { // 使用自建 API (完整功能) loadVercelAPI(); } else if (isGitHub) { // 使用免费计数器服务 loadHitWebCounter(); loadVisitorBadge(); } else { // 本地开发使用本地存储 loadLocalCounter(); } } // HitWebCounter 集成 function loadHitWebCounter() { const img = document.createElement('img'); img.src = 'https://hitwebcounter.com/counter/counter.php?page=YOUR_ID&style=0006&nbdigits=5&type=page&initCount=0'; img.alt = 'Visit Counter'; img.style.border = 'none'; document.getElementById('counter-container').appendChild(img); } // Visitor Badge 集成 function loadVisitorBadge() { const img = document.createElement('img'); img.src = 'https://visitor-badge.laobi.icu/badge?page_id=your.page&left_color=gray&right_color=blue&left_text=Visitors'; img.alt = 'Visitor Badge'; document.getElementById('badge-container').appendChild(img); }

🚀 快速实施步骤

  1. 选择主要服务：推荐 HitWebCounter (功能丰富)
  2. 获取代码：访问 hitwebcounter.com 获取您的计数器代码
  3. 集成到页面：将代码添加到您的 footer 组件中
  4. 添加备用方案：集成 Visitor Badge 作为备用显示
  5. 测试各环境：确保在不同环境下都能正常工作

💡 实施建议

我建议您采用这个混合方案，这样可以：

  • ✅ Vercel 环境：继续使用您现有的高级功能
  • ✅ GitHub Pages：使用 HitWebCounter + Visitor Badge
  • ✅ 本地开发：使用本地存储计数
  • ✅ 备用降级：多层备用确保总有计数显示

这样既保持了功能的完整性，又大大简化了维护工作！
