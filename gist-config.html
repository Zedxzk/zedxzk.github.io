🔧 Gist计数器配置

说明：为了使用GitHub Gist访问计数功能，您需要提供一个GitHub Personal Access Token。 Token仅存储在您的浏览器本地，不会发送到任何服务器。

如何获取GitHub Token？

  1. 访问 GitHub Settings → Personal access tokens
  2. 点击 "Generate new token (classic)"
  3. 设置名称，如 "Website Gist Counter"
  4. 选择权限：只需要勾选 gist
  5. 点击 "Generate token"
  6. 复制生成的token并粘贴到上方输入框
